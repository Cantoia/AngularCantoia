# angular-dashboard
Angular Dashboard Demo

## Yeoman

```bash
$ yo angular
```

## Setup

### Windows
```bash
$ npm install && bower install
$ npm install -g http-server
$ cd app && bower install
$ http-server
```

Acceder a [http://localhost:8080/#/](http://localhost:8080/#/)

### GNU/Linux
```bash
$ npm install && bower install
$ grunt serve
```
